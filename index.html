Administración de PyMES :
Aprenderás a realizar planillas desde cero, aplicar formatos y como utilizar formulas y funciones para automatizar algunos procesos, está modalidad se divide en dos módulos en el primer módulo trabajaras con EXCEL y el segundo módulo con ACCESS.


Primer módulo: Microsoft EXCEL:

Excel es un programa muy utilizado en la actualidad y cualquier persona debe aprender a utilizarlo, pero ¿porque debería aprender a utilizarlo? Seguramente has visto que gran parte de las personas todavía siguen realizando planillas en papel, siendo que hoy se puede digitalizar, por ejemplo: antes si queríamos saber en qué mes se obtuvo la mayor cantidad de ventas teníamos que revisar una por una las planillas, realizar todo eso de forma manual era un trabajo y podía haber errores, en cambio en Excel es mucho más fácil y simple. Siempre y cuando sepamos utilizar las herramientas correspondientes, en esta capacitación serás capaz de poder realizar gráficos estadísticos, planillas de liquidaciones de sueldos, control de stock, podrás realizar balances entre muchas otras cosas.



Segundo módulo: Microsoft Access:

Access es un programa utilizado para poder gestionar bases de datos, ahora, si te preguntas ¿qué es una base de datos? Esa pregunta la vas a poder responder cuando finalices tu capacitación. Mientras tanto en Access aprenderás como se realiza una base de datos, en que consiste y como obtener información de ella, también sabrás como generar formularios para carga de datos, así como informes entre otras muchas cosas.
